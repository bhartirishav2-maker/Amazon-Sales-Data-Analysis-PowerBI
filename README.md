# Amazon E-commerce Sales Analysis Dashboard

## Project Overview
This project involves a comprehensive analysis of Amazon sales data using Power BI. The goal was to transform raw data into a professional, interactive dashboard that provides actionable business insights.
## Key Features
* **KPI Tracking:** Real-time visibility into Revenue ($18M), Profit ($5M), and Units Sold (137K).
* **Trend Analysis:** Monthly Revenue & Profit trends showing growth patterns.
* **Product Performance:** Top 10 products ranked by total sales volume.
* **Geographic Insights:** Map visualization of sales distribution across North, South, and West regions.
* **Interactive Slicers:** Dynamic filtering by Category, Year, and Region.

## Technical Skills Used
* **Data Cleaning:** Used Power Query to handle null values and filter out "TOTAL" rows from raw Excel data.
* **Data Modeling:** Established relationships between Category, Monthly, and Region summary tables.
* **DAX:** Created custom measures for Revenue and Profit Margin analysis.
* **Visualization:** Applied advanced formatting, including shadows, custom themes, and conditional maps.
